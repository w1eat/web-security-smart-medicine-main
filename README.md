# web-security-smart-medicine-main

### 开发工具与环境

- **开发工具**

    - IntelliJ IDEA 2020.2.2：主要进行系统的开发、系统调试等
        - 使用到的插件
            - Lombok：使用注解代替在实体类中添加的 get/set/toString 方法等
    - WebStorm 2019.2.1：主要进行前端页面的开发
    - Navcat：主要进行数据库的连接、建库建表、系统调试等

- **开发环境**

    - JDK 1.8
    - Maven 3.6
    - MySQL 5.7
### 导入数据库
导入smart-medicine.sql到创建的数据库中。
修改配置文件\src\main\resources\application.yml为自己的配置
